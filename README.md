## üõ†Ô∏è Month 1: The Foundation Phase (Weekly Schedule)

### Week 1: Environment & Mindset
- [ ] Install [VirtualBox](https://www.virtualbox.org/) or VMware.
- [ ] Download and install [Kali Linux](https://www.kali.org/get-kali/).
- [ ] Complete **TryHackMe:** [Tutorial Room](https://tryhackme.com/room/tutorial).
- [ ] Complete **TryHackMe:** [Intro to Cybersecurity Path](https://tryhackme.com/path/outline/introtocyber).

### Week 2: Linux Mastery
- [ ] Play **OverTheWire:** [Bandit (Levels 0-10)](https://overthewire.org/wargames/bandit/).
- [ ] Learn basic CLI: `ls`, `cd`, `pwd`, `grep`, `nano`, `chmod`.
- [ ] **Project:** Create a shell script that creates a folder and a text file inside it automatically.

### Week 3: Networking Fundamentals
- [ ] Understand the **OSI Model** (Focus on Layers 2, 3, 4, and 7).
- [ ] Learn **IP Addressing & Subnetting** (Private vs Public).
- [ ] Complete **TryHackMe:** [Network Fundamentals](https://tryhackme.com/module/network-fundamentals).
- [ ] **Lab:** Open [Wireshark](https://www.wireshark.org/) and capture your own browser traffic.

### Week 4: Web Architecture
- [ ] Learn how **DNS** works (Domain Name System).
- [ ] Study **HTTP/HTTPS** Request/Response cycles and Status Codes.
- [ ] Complete **TryHackMe:** [How the Web Works](https://tryhackme.com/module/how-the-web-works).

## üêç Python for Security (Mini-Roadmap)
*Spend 30 mins of your daily 2 hours on this starting in Month 4.*
1. **Basics:** Variables, Loops, Functions.
2. **Libraries:** `requests` (for web), `scapy` (for packets), `socket` (for connections).
3. **Project 1:** Build a simple **Port Scanner**.
4. **Project 2:** Build a **Keylogger** (for educational purposes).
5. **Project 3:** Build a **Password Cracker** (Dictionary attack).
